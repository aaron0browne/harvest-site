Technical Documentation

ModelTree: the "engine" behind Avocado which contains adds an additional layer to the Django ORM for auto-joining database tables

Serrano: a RESTful API exposing various parts of Avocado allows for programmatic access to the metadata

Avocado: the metadata management app and extension to ModelTree which defines the organizational data structures such as Definition, Concept, and Domain

Cilantro: an in-browser Web application interface which talks to the Serrano exposed REST API for graphical querying and viewing data
