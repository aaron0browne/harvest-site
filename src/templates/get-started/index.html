What is Harvest?

Harvest is an open source framework for developing web applications for biomedical data discovery. Harvest grew out of our desire to create purpose-built data discovery applications for multiple projects while avoiding the pitfalls of trying to build a large, monolithic application that attempts to be all things to all people.

Harvest is not a "shrink-wrapped" piece of software in the sense that it can be installed and configured by the average end-user. Instead, it is designed to be installed and customized by software developers and informatics personnel. For some uses, a little bit of programming will be needed to develop custom components for your specific scenario. However, even out of the box, you should be able to adapt Harvest to a variety of scenarios with little trouble. Our goal is that as Harvest is used in more and more projects, additional components will be made available to the community that address common needs.

Brief Technical Overview

Harvest relies heavily on the popular Python Django web framework. Knowledge of Django is important for really understanding Harvest. However, Django is renowned for its excellent documentation, so it's possible to learn Harvest and Django at the same time. The Harvest Stack is composed of three core components.

Avocado

Avocado is without a doubt the most important part of the framework. It provides the metadata abstraction layer that allows Harvest applications to hide the complexity of the underlying relational database models. The tutorials show how some of this works, but many of the technical details can be found in the standalone Avocado documentation.

Serrano

Serrano is a REST API that exposes URL endpoints to enable clients to consume the data encapsulated by Avocado. Additionally, Serrano is responsible for managing persistence for Harvest applications. In addition to standard HTML, Serrano is able to provide alternate data representations and formats such as those required for Excel, SAS, and R. Serrano's API is fully documented to encourage the development of fit for purpose Harvest clients (e.g. native mobile applications) that provide some or all of the functionality provided by the standard Cilantro client (below) that ships with Harvest.

Cilantro

Cilantro is an HTML5/JavaScript client that generates and displays histograms, bar charts, and pie charts for suitable database fields in real time, allowing users to see a profile of specific data elements even before constructing formal queries. This is the default user interface for Harvest that is designed to work well for a majority of typical data discovery use cases. For more information on the technical details, see the Cilantro developer docs.
